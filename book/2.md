# Element yapısı

HTML aslında sadece bir çok elementden oluşur. Yani, JavaScript veya Python'un aksine HTML'de bir mantık veya akış yoktur, HTML statiktir. Bu yüzden aslında HTML'in bir programlama dili olmadığını söyleyebiliriz

HTML'de element yazmak için **hem bir etikete hem de içeriğe** ihtiyacımız vardır

## Etiket

Etiket, elementi tanımlayan işaretleme kodudur. İki kısımdan oluşur

açılış etiketi

```html
<html>
```

ve bir kapanış etiketi

```html
</html>
```

aradaki kısım da içeriktir. Farkettiyseniz etiketlerin başında küçüktür işareti (`<`) ve sonunda büyüktür (`>`) işareti var. Bu, HTML'e "burası etiket" der. Kapanış etiketinin başında da slash (`/`) eklenirse kapanış etiketi olur

```html
<p>Merhaba Dünya</p>
```

## Kendi Kendine Kapanan Elementler

Bazı elementler hiç içerik almaz, ve bu yüzden de bir kapanma etiketine ihtiyaç duymaz. Bunlara kendi kendine kapanan elementler veya void element denir. En büyük örneği `<br>` etiketidir, bu element kendi kendine kapanır çünkü yaptığı tek şey yeni satıra geçmek, bir içeriğe ihtiyacı yok. Bundan ne kapanma etiketi var ne de içerik kısmı

```html
<br>
```

## Etiket Özellikleri

Etiket'in davranışını değiştirmek için etiket özellikleri kullanılır. Bunlar, etiket hakkında ekstra bilgi verir böylelikle element daha farklı davranır. Özellikler arasında boşluk bırakılır, özellik ile değer arasında eşittir (`=`) işareti olur. Değer kısmı çift tırnak (`"`) içine alınır

```html
<img src="images/image.png">
```

burada **<img>** etiket, ve onun içindeki **src="images/image.png"** ise özelliktir. Farklı özellikler arasında boşluk bırakılır

```html
<img src="images/image.png" alt="Resim yüklenemedi" width="300">
```
